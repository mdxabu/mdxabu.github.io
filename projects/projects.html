Llama-NL2SQL: Leveraging Open-Source AI Models for Natural Language to SQL Query Translation

Llama-NL2SQL is an open-source project that leverages the LLama AI model to convert natural language queries into SQL statements. It supports various SQL databases and simplifies database interactions by transforming human-readable queries into structured SQL commands.

  • Convert natural language to SQL
  • Powered by Open Source LLama AI
  • Compatible with MySQL databases
